# Note

关于代码注释

主要的说明的类注释

类注释 类注释的设置相比来说较为简单，首先使用快捷键ctrl+alt+s弹出Settings对话框，选择Editor里面的File and Code Templates，在Includes里面选择File Header选项：

在右侧的编辑框里面即可自定义自己想要呈现的注释内容，此处我只填写了作者和日期：

/**
 * @author: ${USER} 
 * @date: ${DATE}
 */
1
2
3
4

这样设置之后，每次新建类时便会自动显示作者和当前日期，当然此处也可以添加其他内容。
